# Console Task Master

A console-based task manager for organizing personal tasks with statuses, priorities, and deadlines.

## Features
- Add, delete, and view tasks
- Change task status (NEW, IN_PROGRESS, DONE)
- Change task priority (LOW, MEDIUM, HIGH)
- Filter tasks by status, priority, or overdue deadlines
- Data is stored in memory (persistence can be added)
